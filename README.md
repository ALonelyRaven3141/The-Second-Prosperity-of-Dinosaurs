
The Second Prosperity of Dinosaurs
========

このMod制作は私の初めての試みです。OpenAIのCodexとともに、素人がどこまでちゃんとしたMinecraftのModを
作れるのか。そういう実験でもあります。ですが本当の目的は、「作りたいModを作る」ことです。

今のMinecraftにはすでに、化石からDNAを取り出し、現代に復活させられるModがいくつかあります。
しかしそれらは、ただ復活させるだけ、その後の面白みがあまりないものが多い印象です。
このModは、復活させるためのプロセスから、復活させたあとにできることまで、細かなところまで楽しめる、
そんなものにしたいと思っています。
